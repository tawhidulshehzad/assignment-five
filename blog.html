Question 1: What are the different ways to select an element in the DOM?

There are several methods for selecting elements in the DOM. Some methods choose elements based on their IDs, classes, tag names, or CSS selectors. Some return a single element, while others return multiple elements.

Question 2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, and textContent are properties used to retrieve or modify the content of an HTML element, but they differ significantly. innerHTML: Gets or sets the HTML content within an element, including nested tags and formatting. It parses HTML and can insert or return markup-containing elements. innerText: Returns or sets only the visible text within an element, ignoring any hidden elements or formatting. It takes into account styles such as display:none. TextContent: Retrieves or sets all text within an element, including hidden elements, but does not parse or process HTML tags. It returns the text as it appears in the document structure.

Question 3: What is event bubbling in the DOM?

Event bubbling is a behavior in which an event triggered on a child element spreads upwards to its ancestors. This means that an event fired on one element can cause events to be fired on its parent elements unless it is stopped.

Question 4: What is event delegation in the DOM?

Event delegation is a technique that involves a parent element listening for events on its child elements. Instead of attaching event listeners to multiple elements, the event is handled at a higher level via event propagation.

Question 5: How do you create, add, and remove elements using JavaScript?

JavaScript can dynamically manipulate the DOM by creating, adding, and removing elements. New elements can be created and added to the DOM, while existing elements can be removed when they are no longer required.
